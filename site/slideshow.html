GitOps-First Design

The entire platform is defined in Git. One bootstrap command, zero drift.

App-of-Apps Pattern

Wave 0 — Namespaces, RBAC, GitOps config
Wave 1 — Vault, ESO, Keycloak, AAP, Pipelines
Wave 2 — Auth, SSO, Secret Stores
Wave 3 — DevSpaces, Shared Builds, User Projects

What This Means

  • Entire platform in version control
  • Consistent across environments
  • Disaster recovery = redeploy from git
  • Changes are PRs, not tickets
  • Audit trail built in
