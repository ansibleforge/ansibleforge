04  —  The Manual Toll
No automation, no shortcut

Walk to the box. Repeat 8.5 million times.

Most organizations had no automated path. Recovery was a four-step loop, performed by a human being, on every affected machine — in a data center, on a desk, on a kiosk.

Physically reach the host — or talk a user through it on the phone.

03 · Remove

Delete the bad driver

Navigate the file path. Delete the offending .sys. Hope nothing else broke.

Model A · Shared Service01

State runs it. Everyone consumes.

For entities with no IT staff

State operates the Controller and Hub centrally. Subordinate entities log in, launch signed templates against their own infrastructure. They never run the platform.

  • Zero infra burden on the consumer
  • Vetted content, RBAC'd per entity
  • Best fit for libraries, small towns, special districts
Model B · Federated Hub02

State signs. They pull.

For entities running their own ops

State maintains a signed Automation Hub. Subordinate entities run their own lightweight Controller and pull vetted, signed content. Like a CDN for trusted playbooks.

  • Sovereignty preserved — no domain join
  • Air-gap and DMZ-friendly
  • Best fit for counties, K–12, universities, tribes
Model C · Crisis Response03

You enroll. We push the fix.

For when the next 78 minutes hits

State CISO team authors a signed remediation playbook the moment a 0-day lands. Enrolled entities receive it through the federated hub and execute — hours, not weeks.

  • One incident response, statewide
  • Audit trail of who patched, when
  • Force-multiplies a small state IR team
